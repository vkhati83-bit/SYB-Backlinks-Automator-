## How to Use This File

**Option 1: Google Slides / PowerPoint**
Copy each slide's content into a new slide. Use a minimal template with:
- Dark text on white background
- Sans-serif font (like Inter, Helvetica, or Arial)
- Consistent margins

**Option 2: Reveal.js**
This markdown can be converted to a web-based presentation using reveal.js.

**Option 3: Pitch.com / Canva**
Use the content as a guide to build slides in a visual tool.

**Design Tips:**
- Keep backgrounds clean (white or very light gray)
- Use one accent color (SYB brand color)
- Large text, minimal words per slide
- Let the content breathe with whitespace
